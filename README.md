# 💧 image-watermark-html5

**純 HTML5 + JavaScript 圖片浮水印工具**

一個專為**身分證件等敏感文件**設計的圖片浮水印工具，強調**本機執行**、**無需安裝**、**正體中文介面**，提供一個安全、簡便的浮水印添加方案。

## 🚀 專案簡介

`image-watermark-html5` 是一個輕量級的圖片浮水印產生器。它允許使用者直接在**瀏覽器端**為圖片添加文字浮水印，處理過程完全在您的**本機設備**上完成，**不會上傳**任何圖片到伺服器，讓您能安心地為如身份證件、合約文件等敏感資料加上「僅供某用途」的保護標記。

---

## 🛠️ 修改緣起與特色

本專案 Fork 自優秀的 [unilei/image-watermark-tool](https://github.com/unilei/image-watermark-tool)（Vue.js 版本），並進行了以下關鍵修改：

### 🌟 核心特色：安全與便利

1.  **【極高安全性】純本機瀏覽器操作：**
    * 專案的核心邏輯已從 Vue.js 轉寫為純 HTML5 + JavaScript。
    * 這確保了圖片從載入、添加浮水印到下載的**整個過程都在您的瀏覽器中完成**。
    * **您的圖片（特別是身份證件）不會被傳輸到任何外部伺服器**，大大提高了處理敏感文件的安全性。

2.  **【極簡使用】無需安裝，開箱即用：**
    * 只需下載整個專案，直接用瀏覽器開啟 `index.html` 檔案即可使用，無需任何 Node.js 或其他環境安裝。

3.  **【在地化】提供正體中文介面：**
    * 介面和說明文字皆採用**正體中文**（臺灣慣用中文），提升在地使用者體驗。

4.  **【高相容性】現代瀏覽器支援：**
    * 基於標準 HTML5 Canvas API 實現，支援主流現代瀏覽器。

## 💻 如何使用

使用本工具非常簡單，**無需網路連線**！

1.  **下載專案：**
    * 點擊右上角的綠色 `<> Code` 按鈕，選擇 **`Download ZIP`** 下載整個專案。

2.  **執行：**
    * 解壓縮下載的 ZIP 檔案。
    * 用您慣用的瀏覽器（如 Chrome, Firefox, Edge, Safari...）**開啟資料夾中的 `index.html` 檔案**。

3.  **操作步驟：**
    * 點擊按鈕載入您要處理的圖片。
    * 在輸入框中設定浮水印文字（例如：`僅供網路報稅使用，他用無效`）。
    * 調整字體大小、顏色、透明度和浮水印間距，在預覽區即時查看效果。
    * 滿意後點擊 **下載圖片** 按鈕，即可獲得帶有浮水印的圖片檔案。
